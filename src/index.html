Verron Jewelry

Une plateforme e-commerce avec Next.js pour la vente de bijoux. L’application présente un catalogue de produits avec une interface moderne, responsive et facile à utiliser. Platforme de gestion des cellules de réseau télécom pour l'opérateur DJEZZY avec historique

Plateforme web conçue pour Optimum Telecom Algérie (Djezzy) permettant la gestion des cellules réseau avec des utilisateurs aux rôles distincts (admin, éditeur, visionneur) et des opérations CRUD sur les cellules et les utilisateurs. Elle comprend une carte interactive pour les emplacements précis des cellules et un historique des modifications. Réalisée avec Node.js (Express), React.js et MySQL.

SRI classique basé sur le model booléen standard

Une application de bureau permettant la recherche d’informations dans un corpus de documents en utilisant le modèle booléen. Développée en Python avec la bibliothèque NLTK pour le traitement du langage et MongoDB pour le stockage des données. Le projet combine l’algorithme de Porter pour la racinisation, un index inversé pour l’indexation des termes, et une interface utilisateur pour l’affichage des résultats de recherche.
